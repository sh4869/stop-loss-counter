# カウンターアプリ

シンプルなカウンターアプリケーションです。数値の増減と操作履歴の記録ができます。

## 機能

- ✨ **カウンター機能**: `+` と `-` ボタンで数値を増減
- 📝 **履歴表示**: 過去5回の操作履歴を表示
- 💾 **データ永続化**: ページをリロードしてもデータが保持されます
- 🗑️ **履歴クリア**: ボタン一つで履歴とカウント値をリセット

## 技術スタック

- React 18
- TypeScript 5
- Vite
- Tailwind CSS

## セットアップ

### 必要要件

- Node.js 16以上
- npm または yarn

### インストール

```bash
# 依存関係のインストール
npm install

# または
yarn install
```

### 開発サーバーの起動

```bash
npm run dev

# または
yarn dev
```

ブラウザで `http://localhost:5173` を開いてください。

### ビルド

```bash
npm run build

# または
yarn build
```

ビルドされたファイルは `dist` ディレクトリに出力されます。

### プレビュー

```bash
npm run preview

# または
yarn preview
```

## 使い方

1. `+` ボタンをクリックすると数値が1増加します
2. `-` ボタンをクリックすると数値が1減少します
3. 操作履歴は画面下部に最新5件まで表示されます
4. 「履歴をクリア」ボタンで全てのデータをリセットできます
5. ページをリロードしてもデータは保持されます
